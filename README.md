
# Aplikasi EZLife
Aplikasi ini merupakan aplikasi yang berisi informasi pribadi dari saya sendiri yang mengembangkannya. Terdapat beberapa menu di dalamnya antar lain, kontak, kalkulator, biodata, cuaca, dan berita lucu.
